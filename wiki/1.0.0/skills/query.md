# Wiki Query

Query wiki pages and synthesize answer.

## When to Use

When user types `wiki:query <question>` or asks a question about wiki content.

## The Process

1. **Read INDEX**
   - Read workspace/knowledge-base/wiki/INDEX.md
   - Identify relevant sections and pages for the question

2. **Read relevant pages**
   - Read the identified wiki pages
   - Extract relevant information

3. **Synthesize answer**
   - Combine information from multiple pages
   - Use [[Page Name]] format for citations
   - Provide clear, concise answer

4. **Offer to save**
   - If answer is valuable and not already in wiki
   - Suggest creating new wiki page with Russian name

## Output

Answer with citations, optionally offer to save as new page.
